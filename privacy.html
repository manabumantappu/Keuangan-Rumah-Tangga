Kebijakan Privasi

Aplikasi Keuangan Rumah Tangga Islami tidak mengumpulkan, menyimpan, atau membagikan data pribadi pengguna.

Data Pengguna

Semua data keuangan disimpan secara lokal di perangkat pengguna menggunakan penyimpanan lokal (local storage) dan tidak dikirim ke server mana pun.

Iklan

Aplikasi ini tidak menampilkan iklan.

Akses

Aplikasi dapat digunakan tanpa login atau pembuatan akun.

Perubahan

Kebijakan privasi ini dapat diperbarui jika diperlukan.
